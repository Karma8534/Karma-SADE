## Key Learnings (Don't Repeat)

1. **Docker volume mounts must be explicit** — Files hardcoded in container won't exist unless mounted
2. **Commit incrementally to main** — Each session commits work immediately, not at end
3. **K2 infrastructure must be locked down** — No reconfiguration each session
4. **Test connectivity first** — Before building write-back, verify read works
5. **One blocker per session** — Don't spin on multiple unknowns, pick one and finish it
