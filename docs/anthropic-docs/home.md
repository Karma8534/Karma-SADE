## Recommended path for new developers

2. **Understand the Messages API** — Core request/response structure, multi-turn conversations, system prompts, stop reasons. [Messages API guide](https://platform.claude.com/docs/en/build-with-claude/working-with-messages)
3. **Choose the right model** — Compare Claude models by capability and cost. [Models overview](https://platform.claude.com/docs/en/about-claude/models/overview)
4. **Explore features and tools** — Extended thinking, web search, file handling, structured outputs, and more. [Features overview](https://platform.claude.com/docs/en/build-with-claude/overview)
