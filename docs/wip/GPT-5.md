FEBRUARY 9, 2026 • 1 MINUTE READ
GPT-5.3-Codex is now generally available for
GitHub Copilot
GPT-5.3-Codex, OpenAI’s latest agentic coding model, is now rolling out in
GitHub Copilot. In early testing, GPT-5.3-Codex reaches new high scores on
benchmarks we use to evaluate coding, agentic, and real-world capabilities. It
also has improved reasoning and execution in complex, tool-driven, long-running
workflows which help it to deliver up to 25% faster performance than GPT-5.2-
Codex on agentic coding tasks.
